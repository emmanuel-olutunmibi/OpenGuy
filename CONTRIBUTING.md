# Contributing to OpenGuy

Thank you for your interest in contributing to OpenGuy!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/OpenGuy.git`
3. Create a branch: `git checkout -b your-feature-name`
4. Make your changes
5. Commit: `git commit -m "Description of your changes"`
6. Push: `git push origin your-feature-name`
7. Open a Pull Request

## Areas to Contribute

- Kinematics and motion planning (Python)
- Hardware drivers and C integration
- Robot simulation improvements
- Tests and documentation
- Bug fixes

## Code Style

- Follow PEP8 for Python code
- Add docstrings to all functions
- Write tests for new features

## Questions?

Open an issue on GitHub or reach out via the community channels.
